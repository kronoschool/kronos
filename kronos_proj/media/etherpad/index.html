How to use

The original embed idea comes from the etherpad blog.
You can get the text source via this link: http://etherpad.com/ep/pad/export/$ep/latest?format=txt.
So basically we can use etherpad as a black box, off site service. Each lesson gets assigned a "pad" and at the end of the session the source gets copied into our system and the ethepad server could not exist for all we know...
We can have an etherpad for ANY TextArea in fact.... tries to lead a collab verison of the text area if it fails then it just loads a regular textarea form.
